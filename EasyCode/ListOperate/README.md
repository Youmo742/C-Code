# 链表的学习笔记
## 顺序表
  ###  顺序表是计算机中以数组形式存储的线性表，其要求是**逻辑连续**的同时,**物理空间**也连续。
```C++
#define MaxSize 64
typedef int ElemType

typedef struct Node
{
  ElemType[MaxSize];
  Elemtype len;
}
```
**这便定义一个是顺序表**
#### 1 顺序表的优点  
a. 查找方便，随机存取数据  
b. 空间利用率高，存储效率高 
#### 2 顺序表的缺点
a. 其容量的大小提前确定，容易造成空间的浪费或缺乏.
b. 空间不易扩充，插入或删除麻烦，插入或删除需要移动大量元素，时间复杂度为  O（n）
